Данный скрипт создает бекап определенной директории и отправляет его до удаленного сервера, где предвартиельно настроен доступ
по ssh без пароля.


Выберите нужный пункт: 
1)tar- создаст tar.gz архив
2)scp- отправит бекап до заббикс сервера
3)rm- удалит созданный архив на локальном сервере
4)stop- завершение скрипта
