Данный скрипт позволяет снять основную диагностическую информацию о состоянии кластера Континент 4.
1) Проверка работоспособности служб отвечающих за кластеризацию
2) Проверка версии конфигурации
3) Создание архива с логами + файл, в который собрана диагностическая информация о кластере.
4) Как доп. функция добавлены строчки для анализа конфигурации динамической маршрутизации.
