checkusername скрипт проверяющий, есть ли уже пользователь с таким именем
Скрипт useradd создает пользователя и его группу, если такой не существует
Если группа it или security, то пользователь заносится в файл sudoers, также делается бекап данного файла
Затем проверяется, есть ли домашняя папка у данной группы, если её нет, то она создается
Также новому пользователю задается пароль по умолчанию P@ssw0rd, т.к без пароля пользователь не сможет
подлкючиться по ssh к серверу, а затем пароль истекает, чтобы при первом входе пользователь сразу сменил стандартный пароль
